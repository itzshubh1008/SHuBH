## Using GitHub Desktop (Easiest & Most Reliable)

### Step 1: Download GitHub Desktop
- Go to: https://desktop.github.com/
- Download and install
- Sign in with your GitHub account

### Step 2: Create Repository on GitHub
1. Go to https://github.com/new
2. Name it: `portfolio`
3. Make it **Public**
4. Do NOT add README
5. Click **"Create repository"**

### Step 3: Clone Repository
1. Open GitHub Desktop
2. Click **"File"** â†’ **"Clone repository"**
3. Search for your repository name
4. Choose folder to save (e.g., `d:\github\portfolio`)
5. Click **"Clone"**

### Step 4: Copy Your Files
1. Open `d:\mv` (your current portfolio folder)
2. Open the cloned repository folder (from Step 3)
3. **Copy everything from `d:\mv` into the cloned folder:**
   - index.html
   - style.css
   - script.js
   - .gitignore
   - profile/ (entire folder)
   - certificates/ (entire folder)
   - All .md files

### Step 5: Commit and Push
1. Go back to **GitHub Desktop**
2. You'll see all files listed
3. Bottom left, enter message: `Initial portfolio commit`
4. Click **"Commit to main"**
5. Click **"Push origin"** (top right)
6. Done! âœ…

### Step 6: Enable GitHub Pages
1. Go to: https://github.com/YOUR_USERNAME/portfolio/settings
2. Scroll to **"Pages"**
3. Select **main** branch
4. Select **/root** folder
5. Click **"Save"**

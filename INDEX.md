2. Find the sections you want to change
3. Replace example content with yours
4. Save the file

**Start with:** Hero section, About, Contact info

### Step 2: Test Locally (5 min)
1. Open `index.html` in your web browser
2. Check that everything displays correctly
3. Test all links and buttons
